# Loom
Bunch of utilities for programming in C.

# Features
- Data Structures
    - [ ] String
    - [ ] String View
    - [ ] Array
    - [ ] Dynamic Array
    - [ ] HashMap

- Allocators
    - [ ] Arena
    - [ ] Slab
    - [ ] TLSF

- [ ] Performance Profiling Utilities

# Build and Test
- Build examples and tests
```bash
make MODE=release -j$(nproc)
```

- Build and run tests
```bash
make run
```
